# AirBnB-Case-Study
Airbnb is a tech company that enables hosts to rent out their homes, apartments, or rooms to guests interested in temporary lodging. One of the key challenges hosts face is optimizing the rent prices for the customers.

Problem Statement:
Using Machine Learning, predict the optimal Airbnb listing price in NYC given listing information (e.g. bedrooms, type of bed, location, ratings, etc.)

Data Description:
1. host_id - unique id for each host
2. host_name - name of the host
3. city - city in which the rental home is located
4. state - state in which the home is located
5. zipcode - zipcode of the place
6. country - country in which the home is located
7. latitude - latitude of the place
8. longitude - longitude of the place
9. property_type - type of the property given for rent
10. room_type - type of the room
11. accommodates - accomodation for number of people
12. bathrooms - total number of bathrooms
13. bedrooms - total number of bedrooms
14. beds - total number of beds
15. bed_type - type of bed
16. square_feet - area size in square feet
17. price - rental price of the home
18. availability_365 - availability of the home
19. number_of_reviews - reviews given to each home
20. review_scores_rating - ratings for each review
21. review_scores_cleanliness - score given on the cleanliness
22. review_scores_location - score given to the location
23. review_scores_value - score of the value of the home
